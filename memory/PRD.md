# CodEge Landing Page - PRD

## Original Problem Statement
Build a modern, high-converting single-page landing website for a software automation expert (CodEge). Full Turkish UI content with dark mode support, rich animations, and floating Telegram contact button.

## User Personas
- **Small Business Owners**: Looking to automate manual operational tasks
- **Startups**: Want to scale quickly with automated systems
- **High-traffic Platforms**: Need reliable automation for transactions

## Core Requirements (Static)
- Single-page landing site in Turkish
- Dark mode default with light mode toggle
- Responsive design (mobile-first)
- Rich animations (Framer Motion)
- Floating Telegram button
- Sticky header with CTA
- 10 main sections as specified
